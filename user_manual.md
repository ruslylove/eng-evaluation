### คณะวิศวกรรมศาสตร์ มหาวิทยาลัยเทคโนโลยีพระจอมเกล้าพระยอมเกล้าพระนครเหนือ (มจพ.)

## 2. ขั้นตอนที่ 1: ตรวจสอบข้อมูลส่วนบุคคล
เมื่อล็อกอินสำเร็จ ระบบจะนำมาที่ **ขั้นตอนที่ 1** เพื่อตรวจสอบโปรไฟล์ของคุณที่ดึงมาจากฐานข้อมูลบุคลากรโดยอัตโนมัติ

1. **ข้อมูลทั่วไป:** ตรวจสอบคำนำหน้า ชื่อ-นามสกุล ภาควิชา และตำแหน่งทางวิชาการ
2. **ประเภทบุคลากร (Staff Type):** ตรวจสอบว่าเลือกถูกต้องหรือไม่ (เช่น ข้าราชการ / พนักงานมหาวิทยาลัย)
3. **วันที่ได้รับการแต่งตั้งตำแหน่งทางวิชาการ (Appoint Date):** ป้อนวันที่ได้รับการแต่งตั้งฯ เพื่อใช้ในการคำนวณสิทธิ์ย้อนหลังตามข้อบังคับ

```
┌──────────────────────────────────────────────────────────────────────────┐
│                          [พื้นที่สำหรับใส่รูปภาพ]                          │
│                                                                          │
│  - คำแนะนำรูปภาพ: ภาพแถบข้อมูลส่วนบุคคลทางด้านซ้าย (Profile Card)          │
│    และหน้าต่างขั้นตอนที่ 1 ที่มีช่องสำหรับกรอกวันที่แต่งตั้งตำแหน่ง            │
└──────────────────────────────────────────────────────────────────────────┘
ภาพที่ 2.1: หน้าต่างตรวจสอบข้อมูลส่วนบุคคล และส่วนป้อนวันที่แต่งตั้งตำแหน่งทางวิชาการ
```

## 3. ขั้นตอนที่ 2: การเพิ่มข้อมูลผลงานวิชาการและแนบหลักฐาน
ที่ส่วน **ขั้นตอนที่ 2: ผลงานวิชาการที่ยื่นขอ** คุณสามารถระบุรายการผลงานที่ต้องการใช้เสนอขอรับค่าตอบแทนในรอบปัจจุบัน

1. คลิกปุ่ม **"＋ เพิ่มผลงาน"** เพื่อเพิ่มการ์ดผลงานวิจัยรายการใหม่
2. **กรอกรายละเอียดผลงาน:**
   * **ชื่อผลงานทางวิชาการ:** ระบุชื่อเรื่องของผลงาน (ภาษาอังกฤษ/ภาษาไทย)
   * **ส่วนร่วมในผลงาน (Section):** เลือกประเภทข้อกำหนด (เช่น ผลงานที่มีการนำเสนอกรณีที่ 1)
   * **ฐานข้อมูล / ระดับ (Database Tier):** เลือกฐานข้อมูลที่ตีพิมพ์ (เช่น Scopus Q1, TCI 1 เป็นต้น)
   * **บทบาทในผลงาน (Role):** ติ๊กเลือกบทบาทของคุณ (ต้องเป็น First Author หรือ Corresponding Author เท่านั้น)
   * **แหล่งเผยแพร่ (Venue):** ชื่อวารสารวิชาการ หรือชื่องานประชุมวิชาการ
   * **เลข DOI / แหล่งอ้างอิง:** ระบุ URL หรือเลข DOI สำหรับอ้างอิง
3. **การแนบหลักฐาน (Attachments):** อัปโหลดไฟล์หลักฐาน (เช่น PDF ผลงาน, รายงานการจัดอันดับควอไทล์) โดยระบบจะทำการอัปโหลดและจัดเก็บลงใน Google Drive ของคณะโดยแยกโฟลเดอร์รายบุคคลให้อัตโนมัติ (สำหรับแบบตอบรับการตีพิมพ์ สามารถใช้ภาพถ่ายหน้าจอ / Screenshot ของข้อความในอีเมลแทนได้)

```
┌──────────────────────────────────────────────────────────────────────────┐
│                          [พื้นที่สำหรับใส่รูปภาพ]                          │
│                                                                          │
│  - คำแนะนำรูปภาพ: การ์ดป้อนข้อมูลผลงานวิชาการที่มีการป้อนข้อมูลครบถ้วน       │
│    รวมถึงช่องแนบไฟล์หลักฐาน                                                │
└──────────────────────────────────────────────────────────────────────────┘
ภาพที่ 3.1: หน้าต่างฟอร์มกรอกข้อมูลผลงานวิชาการและการอัปโหลดแนบหลักฐาน
```

## 4. ขั้นตอนที่ 3: ตรวจสอบคะแนนสะสมและดูตัวอย่างบันทึกข้อความ
หลังจากป้อนข้อมูลผลงาน ระบบจะทำการเปรียบเทียบกับ **เกณฑ์ภาระงานสะสมขั้นต่ำ** ประจำตำแหน่งของคุณโดยอัตโนมัติ

1. **คะแนนสะสมรวม:** ระบบจะรายงานคะแนนเป็นเปอร์เซ็นต์ (ต้องได้รับอย่างน้อย 100% จึงจะผ่านเกณฑ์)
2. **ดูตัวอย่างบันทึกข้อความ:** คลิกปุ่ม **"📄 ดูตัวอย่างบันทึกข้อความ"**
   * ระบบจะสร้างหน้าพรีวิวบันทึกข้อความในรูปแบบทางการ พร้อมตารางสรุปผลงานและการลงนามของผู้เสนอและหัวหน้าภาควิชา
   * สามารถกดดาวน์โหลดตัวดราฟต์ไฟล์บันทึกข้อความ `.docx` จากหน้านี้ไปเปิดใน Microsoft Word ได้ทันที

```
┌──────────────────────────────────────────────────────────────────────────┐
│                          [พื้นที่สำหรับใส่รูปภาพ]                          │
│                                                                          │
│  - คำแนะนำรูปภาพ: หน้าจอสรุปคะแนนสะสมที่แสดงสถานะสีเขียว (ผ่านเกณฑ์)       │
│    และหน้าต่าง Modal แสดงตัวอย่างพรีวิวบันทึกข้อความราชการ                    │
└──────────────────────────────────────────────────────────────────────────┘
ภาพที่ 4.1: แผงควบคุมแสดงเปอร์เซ็นต์คะแนนสะสม และการแสดงหน้าต่างบันทึกข้อความพรีวิว
```

## 5. ขั้นตอนที่ 4: การยื่นคำขอและบันทึกข้อมูล
หากข้อมูลผลงานและบันทึกข้อความถูกต้องครบถ้วน ให้ดำเนินการยื่นเอกสารเข้าสู่ระบบ

1. คลิกปุ่ม **"ยื่นคำขอ"** ที่ขั้นตอนที่ 3 เพื่อส่งเอกสารไปเก็บในฐานข้อมูลกลาง of คณะ
2. **กรณีที่เคยยื่นในรอบปีนี้ไปแล้ว:** ระบบจะแจ้งเตือนถามว่าต้องการเขียนทับ (Overwrite) หรือไม่
3. เมื่อยื่นสำเร็จ แถบสถานะสีเขียวด้านบนจะแจ้งเตือนว่า **"ยื่นคำขอสำเร็จแล้ว"**

```
┌──────────────────────────────────────────────────────────────────────────┐
│                          [พื้นที่สำหรับใส่รูปภาพ]                          │
│                                                                          │
│  - คำแนะนำรูปภาพ: ปุ่มยื่นคำขอที่มีแถบแจ้งเตือนสถานะสำเร็จ หรือกล่องข้อความ │
│    แจ้งเตือนยืนยันการเขียนทับ                                              │
└──────────────────────────────────────────────────────────────────────────┘
ภาพที่ 5.1: แถบแจ้งเตือนความสำเร็จเมื่อยื่นคำขอเสร็จสิ้น
```

## 6. การตรวจสอบประวัติการยื่นคำขอย้อนหลัง
ผู้ยื่นสามารถเรียกดูรายการคำขอทั้งหมดที่เคยส่งเข้าระบบในอดีตได้ตลอดเวลาผ่านทางเมนูประวัติ

1. คลิกที่แถบนำทาง **"Step 4: ประวัติการยื่นคำขอ"** บริเวณแถบเมนูด้านซ้าย (Sidebar)
2. **ตารางประวัติยื่น:** จะแสดงรอบปีงบประมาณ, รอบที่ยื่น, คะแนนสะสมรวม, และผลการพิจารณาเบื้องต้น
3. **ปุ่ม ดูผลงาน:** คลิกเพื่อเปิดดูรายละเอียดผลงานที่เคยยื่นในรอบนั้นๆ พร้อมลิงก์หลักฐานแนบ
4. **ปุ่ม ดาวน์โหลดบันทึก:** คลิกดาวน์โหลดเอกสารไฟล์บันทึกข้อความราชการ `.docx` ของรอบนั้นๆ เพื่อนำไปสั่งพิมพ์เสนอลงนามได้ทันที

```
┌──────────────────────────────────────────────────────────────────────────┐
│                          [พื้นที่สำหรับใส่รูปภาพ]                          │
│                                                                          │
│  - คำแนะนำรูปภาพ: ตารางแสดงรายการประวัติคำขอย้อนหลังในแท็บประวัติ          │
│    พร้อมภาพปุ่ม "ดูผลงาน" และ "ดาวน์โหลดบันทึก"                             │
└──────────────────────────────────────────────────────────────────────────┘
ภาพที่ 6.1: หน้ารายละเอียดตารางประวัติ และปุ่มดาวน์โหลด/ดูผลงานย้อนหลัง
```

## 7. การแก้ไขปัญหาเบื้องต้น
หากพบปัญหาการใช้งานระบบ สามารถเช็คและแก้ไขเบื้องต้นดังนี้:

* **คำขอหาไม่เจอในประวัติ:** ตรวจสอบให้แน่ใจว่าได้เข้าใช้งานลิงก์ Test Deployment (`/dev`) เพื่อไม่ให้ติดปัญหาแคชไฟล์ HTML เก่าของเบราว์เซอร์
* **ปุ่มดาวน์โหลดเงียบใน Safari:** หากกดดาวน์โหลดบนเบราว์เซอร์ Safari แล้วไม่มีปฏิกิริยา ให้กดปุ่ม **"คลิกที่นี่หากระบบไม่นำทาง"** หรือเปิดสิทธิ์การดาวน์โหลดอัตโนมัติ/อนุญาตป๊อปอัปสำหรับโดเมนของ Google
* **การอัปโหลดไฟล์ไม่สำเร็จ:** ไฟล์หลักฐานที่อัปโหลดต้องมีขนาดไม่เกิน 50MB และจำกัดประเภทเป็นไฟล์ PDF, Word, Excel, หรือรูปภาพมาตรฐานเท่านั้น
